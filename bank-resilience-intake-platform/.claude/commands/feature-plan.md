# Skill: Feature Planning (AI PM Mode)

**觸發時機**：使用者說「我想做XX功能」、「新增」、「加一個」

## 執行步驟

1. **需求釐清**（先問，不急著做）
   - 這個功能解決誰的問題？
   - 使用者的使用場景是什麼？
   - 成功的定義是什麼（怎麼知道做好了）？

2. **範圍定義**
   - MVP：最小可驗證版本需要什麼？
   - 延伸：之後可以加什麼？
   - 排除：這次明確不做什麼？

3. **技術評估**
   - 需要改哪些檔案？
   - 有無相依的現有功能？
   - 預估複雜度：低 / 中 / 高

4. **產出計畫**
```
📋 功能計畫：[功能名稱]
目標用戶：[誰]
使用情境：[何時、如何使用]
MVP 範圍：[具體項目]
影響檔案：[列表]
複雜度：[低/中/高]
建議順序：[步驟 1→2→3]
```

5. **等待確認**後再開始實作
