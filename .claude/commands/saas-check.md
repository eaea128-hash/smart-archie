# Skill: SaaS Readiness Check

## 檢查維度

### 🔴 必須有（阻塞上線）
- [ ] 付費系統（Stripe）設定完成
- [ ] 隱私政策 + 服務條款頁面
- [ ] 錯誤監控（Sentry DSN 已設定）
- [ ] 用戶可以刪除自己的帳號與資料
- [ ] 所有 API 端點有身份驗證

### 🟡 重要（影響轉換率）
- [ ] 新用戶 onboarding 引導
- [ ] 交易型 Email（歡迎信、密碼重設、額度警告）
- [ ] 行動裝置版面正常
- [ ] 404 頁面
- [ ] 載入速度 < 3 秒

### 🟢 加分（成長期）
- [ ] Google/Microsoft OAuth 登入
- [ ] 使用者回饋機制
- [ ] Changelog 頁面
- [ ] API 文件
